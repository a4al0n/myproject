Инструкция по управлению модулями
Как включить/отключить модуль:
  1. Откройте файл shop/settings.py
  2. Найдите секцию MODULES_CONFIG:
MODULES_CONFIG = {
    'reviews': {
        'enabled': True,  # Измените на False для отключения
        'app_name': 'reviews',
    },
    'promotions': {
        'enabled': True,  # Измените на False для отключения
        'app_name': 'promotions',
    }
}
  1. Измените значение enabled на True или False
  2. Перезапустите сервер Django: python manage.py runserver
  3. Обновите эту страницу, чтобы увидеть изменения
Важно:
  • При отключении модуля его URL маршруты не регистрируются
  • Основной функционал магазина продолжает работать без отключенных модулей
  • Данные модулей остаются в базе данных при отключении
